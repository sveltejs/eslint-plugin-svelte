# svelte/no-inspect

> Warns against the use of `$inspect` directive

## :book: Rule Details

This rule reports usages of `$inspect`.

<ESLintCodeBlock>

<!--eslint-skip-->

```svelte
<script>
  /* eslint svelte/no-inspect: "error" */
  // âœ— BAD
  $inspect(1);
</script>
```

</ESLintCodeBlock>

## :wrench: Options

Nothing.

This rule was introduced in eslint-plugin-svelte v2.45.0
